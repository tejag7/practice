Projects

Searching Algorithm for HP Airline Services : In this project we used a search algorithm to process customer requests to fly from some origin to some destination. For back tracking and mapping the source to destination we used the concept of BFS algorithm which uses recursion to search whether HP Airlines serves the requested origin to destination. Coding is done in C++.
Hiding Image In Video : In this project, we proposed data hiding and extraction procedure for AVI (Audio Video Interleave) video embedding the secret message bits in DCT higher order coefficients. The secret information taken here is a gray scale image pixel values. The gray scale pixel values are converted to binary values and embedded those values in higher order coefficient value of DCT of AVI video frames. All experiments are done using MATLAB simulation software.
GESTURES BASED WIRELESS ROBOT GAMING:In this project we developed a robot which was controlled by MEMS accelerometer. Depending on the direction of the MEMS, microcontroller receives different signals and transmits to robot section. Depending on the received data, the microcontroller controls the robot using the motor driver and motors. The wireless communication is done using the RF communication.
